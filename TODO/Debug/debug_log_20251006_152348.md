## Шаг 2: Перенос логики обновления

*   **Действие:** Чтение `LoginActivity.java` и удаление из него вызова `refreshAllContacts`.
*   **Обоснование:** Перенос логики обновления в `ContactsActivity`, чтобы процесс обновления был привязан к экрану, который отображает данные.
*   **Ожидаемый результат:** `LoginActivity` не будет больше инициировать обновление.
