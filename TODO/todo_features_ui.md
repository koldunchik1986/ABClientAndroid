# Улучшения интерфейса экрана входа

На основе предложения пользователя, необходимо улучшить UI/UX экрана `LoginActivity`.

## 1. Задачи

-   [ ] **Реализовать физическое удаление профиля:**
    -   [ ] Добавить кнопку "Удалить" рядом с кнопками добавления и редактирования.
    -   [ ] Перед удалением файла профиля с устройства показывать диалог подтверждения (`AlertDialog`).
    -   [ ] После удаления обновлять список профилей в `Spinner`.

-   [ ] **Заменить текстовые кнопки на иконки:**
    -   [ ] Кнопка "Добавить": иконка "+" (например, `android.R.drawable.ic_input_add`).
    -   [ ] Кнопка "Редактировать": иконка "карандаш" (например, `android.R.drawable.ic_menu_edit`).
    -   [ ] Кнопка "Удалить": иконка "корзина" (например, `android.R.drawable.ic_menu_delete`).
    -   [ ] Убрать текстовые подписи у кнопок.
