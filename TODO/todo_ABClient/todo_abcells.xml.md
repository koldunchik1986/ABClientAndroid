
### 1. План портирования abcells.xml

Файл `abcells.xml` - это XML-файл, содержащий пользовательский "атлас" или справочник по ячейкам карты.

### 2. Функциональность в C#

- **Назначение:** Хранить пользовательские или более точные данные о ячейках карты, которые дополняют или переопределяют данные из основного `map.xml`.
- **Структура:** Содержит набор тегов `<cell>`, каждый из которых описывает одну ячейку и имеет атрибуты:
    - `regnum`: Идентификатор ячейки.
    - `label`: Название локации.
    - `cost`: Стоимость перехода.
    - `visited`: Дата последнего посещения.
    - `verified`: Дата последней проверки.
- **Использование:** Файл загружается методом `LoadAbcMap` в классе `Map` и используется для заполнения словаря `AbcCells`.

### 3. Решение для портирования на Android

Данные из этого файла должны быть перенесены в базу данных Room вместе с остальными данными карты.

### 4. План реализации

План является частью общей стратегии портирования `Map.cs`.

1.  **Создать Entity:**
    - Создать `@Entity` класс `AbcCellEntity` для представления этой таблицы в Room.
2.  **Миграция данных:**
    - При первом запуске приложения, `XmlMapParser` (который нужно будет создать) должен будет прочитать этот файл из `assets`.
    - Данные из каждого тега `<cell>` должны быть сохранены как запись в таблице `abccells` в базе данных Room.
3.  **Доступ к данным:**
    - `MapRepository` будет предоставлять методы для доступа к этим данным (например, `fun getAbcCell(regNum: String)`).
