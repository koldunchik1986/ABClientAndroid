
### 1. План портирования FormShowCookies.cs

Файл `FormShowCookies.cs` реализует простое окно для отображения текущих игровых cookie и их копирования в буфер обмена.

### 2. Функциональность в C#

- **UI:** Окно с текстовым полем (только для чтения) и кнопками "Копировать" и "OK".
- **Логика:**
    - При загрузке формы, она запрашивает у `CookiesManager` строку cookie для домена `www.neverlands.ru`.
    - Отображает эту строку в текстовом поле.
    - Автоматически копирует эту строку в буфер обмена.
    - Кнопка "Копировать" позволяет сделать это повторно.

### 3. Решение для портирования на Android

Это простой утилитарный диалог. Его следует реализовать с помощью `DialogFragment`.

### 4. План реализации

1.  **Создать XML-разметку:**
    - Создать `layout/dialog_show_cookies.xml`.
    - Разместить `TextView` для отображения cookie и `Button` "Копировать".
2.  **Создать класс `ShowCookiesDialogFragment`:**
    - Унаследовать от `DialogFragment`.
3.  **Реализовать логику:**
    - В `onViewCreated`:
        - Получить доступ к `android.webkit.CookieManager`.
        - Вызвать `cookieManager.getCookie("http://www.neverlands.ru")`.
        - Установить полученную строку в `TextView`.
    - В `OnClickListener` для кнопки "Копировать":
        - Получить доступ к `ClipboardManager`.
        - Создать `ClipData` и поместить его в буфер обмена.
        - Показать `Toast` с сообщением "Скопировано".
