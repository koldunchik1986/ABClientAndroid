
### 1. План портирования FormNavigator.Designer.cs

Файл `FormNavigator.Designer.cs` содержит код, автоматически сгенерированный дизайнером Windows Forms. Он описывает внешний вид и расположение элементов управления на форме навигатора.

### 2. Функциональность в C#

- **Назначение:** Инициализировать и настроить все UI-компоненты формы `FormNavigator`.

### 3. Решение для портирования на Android

Этот файл **не портируется напрямую**. Его содержимое служит **визуальным руководством** для создания аналогичного пользовательского интерфейса с помощью XML-разметки в Android.

### 4. План реализации

- [ ] Использовать этот файл как референс для создания `layout/activity_navigator.xml`.
- [ ] Перенести статичный текст в `strings.xml`.
