
### 1. План портирования FormStatEdit.cs

Файл `FormStatEdit.cs` реализует очень простое диалоговое окно для редактирования некой многострочной текстовой информации (вероятно, статистики персонажа).

### 2. Функциональность в C#

- **UI:** Окно, содержащее одно большое текстовое поле (`TextBox`).
- **Логика:**
    - В конструктор передается строка, которая отображается в текстовом поле.
    - В показанном коде отсутствует логика сохранения или возврата измененного текста, но предполагается, что она есть (вероятно, по нажатию кнопки "OK").

### 3. Решение для портирования на Android

Это стандартный диалог для ввода многострочного текста. Его следует реализовать с помощью `DialogFragment`.

### 4. План реализации

1.  **Создать XML-разметку:**
    - Создать `layout/dialog_edit_text.xml`.
    - Разместить `EditText`, установленный в многострочный режим (`android:inputType="textMultiLine"`), внутри `ScrollView`, чтобы обеспечить прокрутку для длинного текста.
    - Добавить кнопки "OK" и "Отмена".
2.  **Создать класс `EditTextDialogFragment`:**
    - Унаследовать от `DialogFragment`.
    - Передавать в него заголовок и начальный текст через `Bundle`.
3.  **Реализовать логику:**
    - В `onViewCreated` установить начальный текст в `EditText`.
    - В `OnClickListener` для кнопки "OK" получить измененный текст из `EditText` и вернуть его через `setFragmentResultListener`.
