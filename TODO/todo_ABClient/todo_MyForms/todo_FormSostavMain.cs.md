
### 1. План портирования FormSostavMain.cs

Файл `FormSostavMain.cs` реализует окно для **отображения** состава группы противников. В отличие от `FormSostav.cs`, эта форма, по-видимому, не предназначена для редактирования.

### 2. Функциональность в C#

- **UI:** Окно, содержащее `ListBox` для отображения списка врагов (`Foe`).
- **Логика:**
    - В конструктор передается строка, содержащая список членов группы, разделенных точкой с запятой.
    - Конструктор парсит эту строку, создает объекты `Foe` и заполняет ими `ListBox`.

### 3. Решение для портирования на Android

Это простой экран для отображения списка. Его следует реализовать как `Activity` или `DialogFragment`.

### 4. План реализации

1.  **Создать XML-разметку:**
    - Создать `layout/dialog_roster_display.xml`.
    - Разместить `RecyclerView` для отображения списка и, возможно, кнопку "Закрыть".
2.  **Создать `RosterDisplayAdapter`:**
    - Создать адаптер для `RecyclerView`, который будет принимать `List<Foe>` и отображать их (например, имя и уровень).
3.  **Создать `RosterDisplayDialogFragment`:**
    - Унаследовать от `DialogFragment`.
    - Принимать строку с составом группы через `Bundle`.
    - В `onViewCreated`:
        - Распарсить строку, создать `List<Foe>`.
        - Создать и установить `RosterDisplayAdapter` для `RecyclerView`, передав ему список.

- [ ] Создать XML-разметку `dialog_roster_display.xml`.
- [ ] Создать `RosterDisplayAdapter` для `RecyclerView`.
- [ ] Создать `RosterDisplayDialogFragment` и реализовать в нем логику парсинга и отображения.
