
### 1. План портирования FormNewPassword.cs

Файл `FormNewPassword.cs` реализует диалоговое окно для создания нового пароля с полем для подтверждения.

### 2. Функциональность в C#

- **UI:** Диалоговое окно с двумя полями для ввода пароля, галочкой "показать пароль" и кнопками OK/Отмена.
- **Логика:**
    - При каждом изменении текста в любом из полей, происходит их сравнение.
    - Кнопка "OK" активна только в том случае, если тексты в обоих полях полностью совпадают.
    - Галочка "показать пароль" одновременно переключает видимость символов в обоих полях ввода.

### 3. Решение для портирования на Android

Это стандартный диалог создания/подтверждения пароля. Его следует реализовать с помощью `DialogFragment`.

### 4. План реализации

1.  **Создать XML-разметку:**
    - Создать `layout/dialog_new_password.xml`.
    - Использовать два `TextInputLayout` с вложенными `TextInputEditText` для основного пароля и его подтверждения.
    - Добавить `CheckBox` для переключения видимости.
2.  **Создать класс `NewPasswordDialogFragment`:**
    - Унаследовать от `DialogFragment`.
3.  **Реализовать логику:**
    - **Переключить видимость:** Добавить слушатель к `CheckBox`, который будет изменять `inputType` у обоих `EditText`.
    - **Проверка совпадения:** Добавить `TextWatcher` к обоим `EditText`. В методе `afterTextChanged` сравнивать `passwordEditText.text.toString()` с `confirmPasswordEditText.text.toString()`.
    - В зависимости от результата сравнения, устанавливать `okButton.isEnabled`.
4.  **Возврат результата:**
    - В `OnClickListener` кнопки "OK" (который будет доступен только если пароли совпадают), вернуть новый пароль через `setFragmentResultListener`.
