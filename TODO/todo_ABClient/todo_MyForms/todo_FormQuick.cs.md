
### 1. План портирования FormQuick.cs

Файл `FormQuick.cs` реализует панель "быстрых действий" - набор кнопок для применения различных атак или предметов на указанного персонажа.

### 2. Функциональность в C#

- **UI:** Небольшое окно с полем для ввода ника цели и набором кнопок (Обычная нападалка, Кровавая, Туман, Яд и т.д.). Есть галочка "Закрывать после действия".
- **Логика:**
    - Каждая кнопка привязана к своему действию.
    - При нажатии на кнопку, вызывается соответствующий статический метод в `FormMain` (например, `FormMain.FastAttack(...)`), которому передается ник из текстового поля.
    - Эти `FastAttack` методы, в свою очередь, запускают механизм автоматизации, проанализированный в `MainPhpFast.cs`.
    - Если установлена галочка "Закрывать после действия", окно закрывается.

### 3. Решение для портирования на Android

Эту панель действий лучше всего реализовать с помощью `BottomSheetDialogFragment`. Это современный компонент, который выезжает снизу экрана и хорошо подходит для контекстных действий.

### 4. План реализации

1.  **Создать XML-разметку:**
    - Создать `layout/bottom_sheet_quick_actions.xml`.
    - Разместить `EditText` для ника и `GridLayout` или `FlexboxLayout` для кнопок. Использовать `ImageButton` с иконками для компактности.
    - Добавить `Switch` для опции авто-закрытия.
2.  **Создать класс `QuickActionsBottomSheet`:**
    - Унаследовать от `com.google.android.material.bottomsheet.BottomSheetDialogFragment`.
    - В `onCreateView` "надуть" разметку.
3.  **Реализовать логику:**
    - В `onViewCreated` назначить `OnClickListener` для каждой кнопки.
    - При нажатии на кнопку:
        - Получить ник из `EditText`.
        - Вызвать соответствующий метод в `FastActionManager` (класс, предложенный в анализе `MainPhpFast.cs`), передав ему тип действия и ник цели.
        - Если `Switch` включен, вызвать `dismiss()`.
4.  **Интеграция:**
    - Вызывать `QuickActionsBottomSheet().show(supportFragmentManager, ...)` из того места, где требуется эта панель (например, из контекстного меню игрока в чате).
