
### 1. План портирования FormCode.Designer.cs

Файл `FormCode.Designer.cs` содержит код, автоматически сгенерированный дизайнером Windows Forms. Он описывает внешний вид и расположение элементов управления на форме для ввода капчи.

### 2. Функциональность в C#

- **Назначение:** Инициализировать и настроить все UI-компоненты формы `FormCode`.
- **Содержимое:**
    - `PictureBox` для отображения картинки с кодом.
    - `TextBox` для ввода кода.
    - `Button` "Обновить".
    - `Button` "Увеличить".
    - `Button` "Ввод".

### 3. Решение для портирования на Android

Этот файл **не портируется напрямую**. Его содержимое служит **визуальным руководством** для создания аналогичного пользовательского интерфейса с помощью XML-разметки в Android.

### 4. План реализации

1.  **Проанализировать UI:** Изучить `InitializeComponent()` для понимания, какие элементы (`PictureBox`, `TextBox`, `Button`) присутствуют на форме.
2.  **Создать XML-разметку:**
    - Создать файл `layout/dialog_captcha.xml`.
    - В нем, используя `ImageView`, `EditText` и `ImageButton` (для кнопок с иконками), воссоздать внешний вид формы.
    - Иконки для кнопок (`_16x16_refresh`, `_16x16_maximize`, `_16x16_wright`) необходимо извлечь из ресурсов проекта и добавить в `res/drawable`.
3.  **Связать с логикой:**
    - Логика, описанная в `FormCode.cs`, будет использовать эту разметку. Этот процесс уже описан в файле `todo_FormCode.cs.md`.

- [ ] Использовать этот файл как референс для создания `layout/dialog_captcha.xml`.
- [ ] Извлечь иконки для кнопок и добавить их в `res/drawable`.
