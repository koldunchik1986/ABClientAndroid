
### 1. План портирования TabType.cs

Файл `TabType.cs` содержит простое перечисление (`enum`) для определения типа вкладки.

### 2. Функциональность в C#

- **Назначение:** Предоставить строго типизированный способ для идентификации содержимого и назначения каждой вкладки.
- **Значения:**
    - `Game`: Основная игровая вкладка.
    - `Forum`: Форум.
    - `PInfo`, `PName`, `PBots`: Различные виды информации о персонажах.
    - `Log`: Лог боя.
    - `Room`: Комната чата.
    - `TodayChat`: Лог сегодняшнего чата.
    - `Other`: Другие веб-страницы.
    - `Notepad`: Вкладка-блокнот.

### 3. Решение для портирования на Android

Этот `enum` должен быть напрямую портирован в `enum class` в Kotlin.

### 4. План реализации

1.  **Создать `enum class TabType`:**
    - Создать файл `TabType.kt`.
    - Объявить в нем `enum class` с соответствующими значениями.
      ```kotlin
      enum class TabType {
          GAME,
          FORUM,
          P_INFO,
          P_NAME,
          P_BOTS,
          LOG,
          ROOM,
          TODAY_CHAT,
          OTHER,
          NOTEPAD
      }
      ```
2.  **Использование:**
    - Этот `enum` будет использоваться в data-классе `TabState` для определения типа каждой вкладки.
