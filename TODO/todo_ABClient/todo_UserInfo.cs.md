# План портирования UserInfo.cs

Файл `UserInfo.cs` определяет простой класс-контейнер для хранения полной информации о персонаже.

## Функциональность в C#

*   **Назначение**: Служит в качестве объекта для передачи данных (Data Transfer Object, DTO). Метод `NeverApi.GetAll()` парсит ответ сервера и заполняет этот объект, который затем используется в других частях приложения (например, в `Contact.Process()`).
*   **Содержимое**: Класс не содержит логики. Он состоит исключительно из публичных полей для хранения всех аспектов состояния персонажа: надетых вещей, эффектов, основной информации (ник, уровень, клан), статусов (в тюрьме, в молчанке), текущей активности (онлайн, локация, бой) и параметров (HP, мана, усталость).

## Проверка на существующую реализацию в Android

- **Результат:** Функциональность не реализована. В Android-проекте отсутствует класс `UserInfo`.

## Решение для портирования на Android

Необходимо создать POJO-класс `UserInfo.java`, который будет в точности повторять структуру C#-класса.
