# План портирования Russian.cs

Файл `Russian.cs` - это статический класс для хранения констант, связанных с русской локализацией.

## Функциональность в C#

*   **Назначение**: Предоставить централизованный доступ к объектам кодировки `windows-1251` и культуры `ru-RU`, которые используются для кодирования/декодирования текста и форматирования данных.
*   **Свойства**:
    *   `Codepage`: `Encoding` для `windows-1251`.
    *   `Culture`: `CultureInfo` для `ru-RU`.

## Проверка на существующую реализацию в Android

- **Результат:** Функциональность полностью реализована и улучшена.
- **Объяснение**:
    1.  В проекте существует класс `app/src/main/java/ru/neverlands/abclient/utils/Russian.java`.
    2.  Этот класс содержит статическое поле `Charset` для `windows-1251`.
    3.  Он также предоставляет удобные статические методы `getString(byte[])` и `getBytes(String)` для выполнения преобразований, что даже лучше, чем просто хранение константы.
    4.  Константа для культуры `ru-RU` вынесена в `AppConsts.java`.
