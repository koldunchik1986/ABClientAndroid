# План портирования HerbCell.cs

Файл `HerbCell.cs` определяет простой класс-модель данных для ячейки карты с травами.

## Функциональность в C#

*   **Назначение**: Хранить информацию о конкретной ячейке карты, в которой есть травы.
*   **Свойства (Properties)**:
    *   `RegNum`: `string` - Регистрационный номер (идентификатор) ячейки.
    *   `Herbs`: `string` - Строка, содержащая названия трав в ячейке.
    *   `UpdatedInTicks`: `long` - Временная метка последнего обновления в тиках (`DateTime.Ticks`).

## Проверка на существующую реализацию в Android

- **Результат:** Функциональность не реализована. В Android-проекте отсутствует класс-модель `HerbCell`.

## Решение для портирования на Android

Необходимо создать POJO-класс `HerbCell.java`, который будет содержать все поля из C#-версии.
