
### 1. План портирования LezBotsClass.cs

Файл `LezBotsClass.cs` представляет собой простой класс данных, описывающий "класс" или тип ботов.

### 2. Функциональность в C#

- **Назначение:** Хранить информацию о типе ботов.
- **Поля:**
    - `Id`: Уникальный идентификатор класса ботов.
    - `Name`: Имя в единственном числе (например, "Крыса").
    - `Plural`: Имя во множественном числе (например, "Крысы").

Этот класс используется в UI, например, для заполнения выпадающего списка при создании новой группы ботов в настройках Автобоя.

### 3. Решение для портирования на Android

Этот класс должен быть напрямую портирован в `data class` в Kotlin.

### 4. План реализации

1.  **Создать `data class LezBotsClass`:**
    - Создать файл `LezBotsClass.kt`.
    - Объявить в нем `data class` с теми же полями.
      ```kotlin
      data class LezBotsClass(
          val id: Int,
          val name: String,
          val plural: String
      )
      ```
2.  **Источник данных:**
    - Данные для этих классов (список всех типов ботов) должны быть загружены из статического источника. Это может быть:
        - JSON-файл в папке `assets`.
        - Предзаполненная таблица в базе данных Room.
