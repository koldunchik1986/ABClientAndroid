
### 1. План портирования abteleports.xml

Файл `abteleports.xml` - это XML-файл, содержащий список всех телепортов на карте.

### 2. Функциональность в C#

- **Назначение:** Хранить статический список локаций, в которых есть телепорт.
- **Структура:** Содержит набор тегов `<teleport>`, каждый из которых имеет атрибуты:
    - `regnum`: Идентификатор ячейки с телепортом.
    - `name`: Название локации телепорта.
- **Использование:** Файл загружается методом `LoadTeleports` в классе `Map` и используется для заполнения словаря `Teleports`. Эта информация критически важна для алгоритма поиска пути, который добавляет ребра между всеми телепортами.

### 3. Решение для портирования на Android

Данные из этого файла должны быть перенесены в базу данных Room вместе с остальными данными карты.

### 4. План реализации

План является частью общей стратегии портирования `Map.cs`.

1.  **Создать Entity:**
    - Создать `@Entity` класс `TeleportEntity` для представления этой таблицы в Room.
2.  **Миграция данных:**
    - При первом запуске приложения, `XmlMapParser` (который нужно будет создать) должен будет прочитать этот файл из `assets`.
    - Данные из каждого тега `<teleport>` должны быть сохранены как запись в таблице `teleports` в базе данных Room.
3.  **Доступ к данным:**
    - `MapRepository` будет предоставлять метод для получения всех телепортов (например, `fun getAllTeleports(): List<TeleportEntity>`), который будет использоваться при построении графа для поиска пути.
