# План портирования Bookmark.cs

Файл `Bookmark.cs` определяет простой класс-модель данных для закладки.

## Функциональность в C#

*   **Назначение**: Хранить информацию об одной закладке (избранной ссылке).
*   **Свойства (Properties)**:
    *   `Title`: `string` - Название закладки, которое видит пользователь.
    *   `Url`: `string` - URL-адрес, на который ведет закладка.
    *   `SmallIcon`: `System.Drawing.Image` - Иконка для закладки.

## Проверка на существующую реализацию в Android

- **Результат:** Функциональность не реализована. В Android-проекте отсутствует класс-модель для хранения закладок.

## Решение для портирования на Android

Необходимо создать POJO-класс `Bookmark.java`, который будет содержать поля из C#-версии. Тип иконки нужно будет адаптировать под Android.
