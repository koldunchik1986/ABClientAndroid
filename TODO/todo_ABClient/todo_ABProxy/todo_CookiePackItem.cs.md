
### 1. План портирования CookiePackItem.cs

Файл `CookiePackItem.cs` представляет собой простую структуру данных для хранения одного cookie (пары имя-значение).

### 2. Функциональность в C#

- **Назначение:** Хранить имя и значение одного cookie.
- **Логика:**
    - Содержит два публичных строковых свойства: `Name` и `Value`.
    - Метод `ToString()` переопределен для отладочного вывода в формате `"Name: Value"`.

### 3. Решение для портирования на Android

Это тривиальный класс. Его можно легко заменить.

### 4. План реализации

- **Kotlin:** Создать `data class CookiePackItem(val name: String, val value: String)`.
- **Java:** Создать простой POJO-класс `CookiePackItem` с двумя полями и конструктором.
- **Альтернатива:** Вообще не создавать этот класс, а использовать `Pair<String, String>` или аналогичную структуру, если не требуется дополнительная логика.

- [ ] Создать data-класс `CookiePackItem` или использовать стандартный `Pair`.
