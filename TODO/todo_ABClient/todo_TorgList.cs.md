# План портирования TorgList.cs

Файл `TorgList.cs` - это статический класс для парсинга правил торга и вычисления цены.

## Функциональность в C#

*   **Назначение**: Обрабатывать специальную строку правил, чтобы определить бонус или скидку для определенной цены товара.
*   **Ключевые методы**:
    *   `Parse(string torgString)`: Парсит строку сложного формата, например `1-10(5),11-20(10)`, и заполняет массив `TorgPair[]`.
    *   `Calculate(int price)`: Вычисляет итоговую цену, находя подходящий диапазон в `TorgPair[]` и применяя соответствующий бонус.
    *   `DoFilter(...)`: Утилита для подстановки значений (цена, название вещи) в строку-шаблон.
