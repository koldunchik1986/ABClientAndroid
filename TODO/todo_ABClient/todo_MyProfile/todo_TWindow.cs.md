# План портирования TWindow.cs

Файл `TWindow.cs` определяет простую структуру для хранения состояния и геометрии окна.

## Функциональность в C#

*   **Назначение**: Хранить информацию о положении, размере и состоянии (свернуто/развернуто) главного окна приложения, чтобы при следующем запуске восстановить его в том же виде.
*   **Реализация**: Является структурой (`struct`).
*   **Свойства**:
    *   `State`: `FormWindowState`
    *   `Left`, `Top`, `Width`, `Height`: `int`

## Проверка на существующую реализацию в Android

- **Результат:** Неприменимо.
- **Объяснение**: Эта структура полностью специфична для оконной среды рабочего стола (desktop). В Android приложения обычно занимают весь экран, и у пользователя нет возможности управлять их положением и размером. Концепция сохранения геометрии окна не имеет смысла.

## Решение для портирования на Android

Никаких действий не требуется.
