# План портирования TSplitter.cs

Файл `TSplitter.cs` определяет простую структуру для хранения состояния разделителя.

## Функциональность в C#

*   **Назначение**: Хранить состояние (ширину и статус "свернут/развернут") для `CollapsibleSplitter`.
*   **Реализация**: Является структурой (`struct`).
*   **Свойства**:
    *   `Width`: `int` - Ширина панели.
    *   `Collapsed`: `bool` - Статус.

## Проверка на существующую реализацию в Android

- **Результат:** Неприменимо.
- **Объяснение**: Эта структура неразрывно связана с `CollapsibleSplitter.cs`, который не портируется. В Android состояние UI-компонентов (их ширина, видимость) получается напрямую из самих `View` или хранится в `ViewModel`, а не в отдельных структурах.

## Решение для портирования на Android

Никаких действий не требуется.
