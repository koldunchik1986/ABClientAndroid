# План портирования TPers.cs

Файл `TPers.cs` определяет простую структуру для хранения настроек и состояния, связанных с персонажем.

## Функциональность в C#

*   **Назначение**: Хранить набор различных параметров, относящихся к персонажу.
*   **Реализация**: Является структурой (`struct`).
*   **Свойства**:
    *   `Guamod`: `bool` - Флаг, активна ли капча.
    *   `IntHP`, `IntMA`: `double` - Вероятно, настройки регенерации HP/MA.
    *   `Ready`: `long` - Метка времени (`Ticks`), когда персонаж будет готов к следующему действию.
    *   `LogReady`: `string` - ID лога боя, к которому относится метка `Ready`.

## Проверка на существующую реализацию в Android

- **Результат:** Функциональность не реализована.

## Решение для портирования на Android

Необходимо создать POJO-класс `TPers.java` и включить его в `UserConfig.java`.
