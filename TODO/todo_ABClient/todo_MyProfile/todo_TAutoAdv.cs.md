# План портирования TAutoAdv.cs

Файл `TAutoAdv.cs` определяет простую структуру для хранения данных авто-рекламы.

## Функциональность в C#

*   **Назначение**: Хранить данные для одного рекламного сообщения: интервал и текст.
*   **Реализация**: Является структурой (`struct`), а не классом, что в данном случае не имеет принципиального значения при портировании.
*   **Свойства**:
    *   `Sec`: `int` - Интервал в секундах.
    *   `Phraz`: `string` - Текст сообщения.
