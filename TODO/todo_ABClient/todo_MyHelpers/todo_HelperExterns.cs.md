# План портирования HelperExterns.cs

Файл `HelperExterns.cs` - это статический класс, содержащий объявления P/Invoke для вызова низкоуровневых функций Windows API.

## Функциональность в C#

*   **Назначение**: Предоставить C#-коду доступ к не-управляемым системным библиотекам Windows (`wininet.dll`, `user32.dll`, `ole32.dll`).
*   **Реализация**: Содержит набор `[DllImport]` атрибутов для тех же самых функций, что и в классе `NativeMethods.cs` (управление кэшем, UI, сетью). Является, по сути, дубликатом или альтернативной версией `NativeMethods.cs`.

## Проверка на существующую реализацию в Android

- **Результат:** Неприменимо. Функциональность на 100% специфична для Windows и не может быть портирована.
- **Объяснение**: Задачи, которые решались этими нативными вызовами, в Android решаются стандартными, высокоуровневыми API.

## Решение для портирования на Android

Никаких действий не требуется. Этот класс не нужно и невозможно портировать.
