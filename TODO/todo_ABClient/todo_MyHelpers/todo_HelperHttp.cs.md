# План портирования HelperHttp.cs

Файл `HelperHttp.cs` - это статический класс с низкоуровневыми вспомогательными методами для работы с HTTP.

## Функциональность в C#

*   **Назначение**: Предоставить утилиты для ручной обработки различных аспектов HTTP-протокола. Функционально является дубликатом класса `ABClient\Helpers\Http.cs`.
*   **Ключевые методы**:
    *   `IPEndPointFromHostPortString`: Парсит строку с хостом и портом.
    *   `DoUnchunk`: Декодирует тело ответа, использующее `Transfer-Encoding: chunked`.
    *   `GZipExpand` / `DeflateExpand`: Распаковывает сжатые GZip и Deflate данные.

## Проверка на существующую реализацию в Android

- **Результат:** Неприменимо. Функциональность не портирована, но в ней нет необходимости.
- **Объяснение**: Современные HTTP-клиенты, включая используемый в проекте `OkHttp`, обрабатывают все эти низкоуровневые детали протокола автоматически.

## Решение для портирования на Android

Никаких действий не требуется. Весь функционал этого класса предоставляется "из коробки" библиотекой `OkHttp`.
