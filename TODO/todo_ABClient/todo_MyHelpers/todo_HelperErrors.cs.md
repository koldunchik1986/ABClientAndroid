# План портирования HelperErrors.cs

Файл `HelperErrors.cs` - это статический класс для генерации HTML-кода для страниц ошибок.

## Функциональность в C#

*   **Назначение**: Собрать стандартный заголовок для HTML-страниц, отображаемых внутри клиента, с CSS-стилями и шапкой.
*   **Ключевые методы**:
    *   `Marker()`: Возвращает HTML-код для цветного баннера с названием приложения.
    *   `Head()`: Генерирует полную секцию `<head>` и открывающий тег `<body>`, включая мета-теги для отключения кэша и встроенные стили.

- **Объяснение**:

    2.  Этот класс в статическом блоке инициализации генерирует точно такой же HTML-заголовок и хранит его в константе.
    3.  Он предоставляет чистые методы `getHead()` и `getMarker()` для доступа к этим данным, а также дополнительный удобный метод `buildPage(String body)`.
