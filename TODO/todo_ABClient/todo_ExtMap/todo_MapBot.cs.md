
### 1. План портирования MapBot.cs

Файл `MapBot.cs` представляет собой простую структуру данных, описывающую одного монстра (бота) на карте.

### 2. Функциональность в C#

- **Назначение:** Хранить информацию о типе бота, который может появиться в определенной ячейке карты.
- **Поля:**
    - `Name`: Имя бота.
    - `MinLevel`: Минимальный уровень бота.
    - `MaxLevel`: Максимальный уровень бота.
    - `C`, `D`: Некие внутренние свойства, загружаемые из `map.xml`. Назначение неясно из кода.

### 3. Решение для портирования на Android

Это тривиальный класс. Его следует портировать как data-класс в Kotlin или простой POJO в Java.

### 4. План реализации

1.  **Создать data-класс `MapBot`:**
    - **Kotlin:**
      ```kotlin
      data class MapBot(
          val name: String,
          val minLevel: Int,
          val maxLevel: Int,
          val c: String, // Назначение полей C и D нужно будет уточнить
          val d: String
      )
      ```
    - **Java:**
      Создать класс `MapBot` с соответствующими полями, конструктором и геттерами.
