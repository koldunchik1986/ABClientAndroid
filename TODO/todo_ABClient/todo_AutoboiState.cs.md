# План портирования AutoboiState.cs

Файл `AutoboiState.cs` определяет перечисление (enum) для состояний автобоя.

## Функциональность в C#

*   **Назначение**: Определяет набор именованных констант, которые представляют различные состояния, в которых может находиться логика автобоя.
*   **Члены перечисления (Enum Members)**:
    *   `AutoboiOff`: Автобой выключен.
    *   `AutoboiOn`: Автобой активен и наносит удары.
    *   `Restoring`: Идет восстановление перед завершением боя.
    *   `Timeout`: Ожидание таймаута боя.
    *   `Guamod`: Ожидание решения капчи (гуамода).

## Проверка на существующую реализацию в Android

- **Результат:** Функциональность не реализована. В Android-проекте отсутствует `enum` или набор констант, описывающий состояния автобоя.

## Решение для портирования на Android

Необходимо создать `enum` в Java с теми же членами, что и в C#-версии. Это обеспечит строгую типизацию и понятность кода, который будет использовать эти состояния.

## План реализации

- [ ] **Создать файл `AutoboiState.java`** в подходящем пакете (например, `ru.neverlands.abclient.model` или `ru.neverlands.abclient.enums`).

- [ ] **Определить `enum`**:
    ```java
    package ru.neverlands.abclient.enums;

    public enum AutoboiState {
        AUTOBOI_OFF,
        AUTOBOI_ON,
        RESTORING,
        TIMEOUT,
        GUAMOD
    }
    ```
    *(Примечание: в Java принято использовать `UPPER_CASE` для членов `enum`)*.
