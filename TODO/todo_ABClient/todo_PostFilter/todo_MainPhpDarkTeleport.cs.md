
### 1. План портирования MainPhpDarkTeleport.cs

Файл `MainPhpDarkTeleport.cs` является частью логики автоматизации, предназначенной для использования способности "Сумеречный телепорт".

### 2. Функциональность в C#

- **Назначение:** Автоматически использовать способность "Сумеречный телепорт".
- **Логика:**
    1.  Ищет в HTML-коде страницы javascript-вызов `abil_1(3, 'vcode')`.
    2.  Извлекает из этого вызова `vcode` (код проверки).
    3.  Если `vcode` найден, генерирует новую HTML-страницу со скрытой формой.
    4.  В форму подставляются все необходимые для использования способности параметры (`useaction`, `addid`, `post_id`, `vcode`).
    5.  Генерирует случайный пункт назначения (`wtelid`) от 1 до 12.
    6.  В конец страницы добавляется javascript-код (`document.ff.submit();`), который автоматически отправляет форму.
    7.  Возвращает сгенерированную HTML-страницу, которая немедленно использует способность.

### 3. Решение для портирования на Android

Эта логика должна быть реализована на стороне Android-приложения, а не в виде javascript-фильтра.

### 4. План реализации

1.  **Создать класс `AbilityManager` в Android:**
    - Этот класс будет отвечать за использование различных способностей.
2.  **Реализовать обработку ответа от сервера:**
    - В `WebViewClient` перехватывать ответы, где может быть доступна эта способность.
    - Если `AbilityManager`'у дана команда использовать телепорт, он должен проанализировать HTML.
3.  **Реализовать логику анализа в `AbilityManager`:**
    - Воспроизвести логику парсинга `vcode` из вызова `abil_1`.
4.  **Выполнение действия:**
    - `AbilityManager` должен сгенерировать случайное число от 1 до 12 для `wtelid`.
    - `AbilityManager` должен сформировать `POST`-запрос на `main.php` со всеми необходимыми параметрами.
    - Выполнить этот запрос с помощью `WebView.postUrl()` или другой сетевой библиотеки.
