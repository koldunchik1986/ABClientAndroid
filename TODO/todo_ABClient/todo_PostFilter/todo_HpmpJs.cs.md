# План портирования HpmpJs.cs

Файл `HpmpJs.cs` - это часть `partial` класса `Filter`, отвечающая за полную замену скрипта `hpmp.js`.

## Функциональность в C#

*   **Назначение**: Заменить стандартный серверный скрипт для отображения HP/MP на кастомную версию, которая дополнительно показывает таймеры до полной регенерации.
*   **Логика**: Метод `HpmpJs` полностью игнорирует входящие данные и генерирует с нуля новый JavaScript-файл. Этот скрипт содержит функции `ins_HP`, `hms` и `cha_HP`. Основная функция `cha_HP` в цикле `setInterval` перерисовывает полоски HP/MP и рассчитывает оставшееся время до их полного восстановления.
