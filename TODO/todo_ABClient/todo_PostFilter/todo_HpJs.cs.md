# План портирования HpJs.cs

Файл `HpJs.cs` - это часть `partial` класса `Filter`, отвечающая за модификацию скрипта `hp.js`.

## Функциональность в C#

*   **Назначение**: Перехватить данные о HP/MA персонажа, которые отображаются в верхнем фрейме.
*   **Логика**: Фильтр находит длинную строку JavaScript-кода, которая отвечает за формирование HTML-кода для полосок здоровья и маны. Он заменяет всю эту строку на один вызов `window.external.ShowHpMaTimers(...)`, передавая в него все параметры (текущее/максимальное HP/MA и т.д.). Это позволяет C#-коду получить полный контроль над отображением этих данных.
