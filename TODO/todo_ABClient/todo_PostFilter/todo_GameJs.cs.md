# План портирования GameJs.cs

Файл `GameJs.cs` - это часть `partial` класса `Filter`, отвечающая за модификацию основного игрового скрипта `game.js`.

## Функциональность в C#

*   **Назначение**: Внедрить в основной скрипт кастомную логику, в основном связанную с автоматическим обновлением фреймов и интеграцией с чатом.
*   **Активная логика**:
    1.  Изменяет размер фрейма с 300 на 400 пикселей.
    2.  Внедряет большой блок JavaScript, добавляющий функции `arenareload` и `toprefresh` для управления автоматическим обновлением верхнего фрейма (`main_top`).
*   **Закомментированная логика**: Содержит большое количество закомментированного кода, который должен был реализовывать глубокую интеграцию с настройками чата (размер, скорость, режим фильтрации) через `window.external`.

## Проверка на существующую реализацию в Android

- **Результат:** Функциональность полностью реализована.
- **Объяснение**:
    1.  В проекте существует класс `app/src/main/java/ru/neverlands/abclient/postfilter/GameJs.java`.
    2.  Метод `process()` в этом классе полностью повторяет **активную** логику C#-версии: он выполняет замену размера фрейма и внедряет тот же самый JavaScript-блок для "AutoArena".
    3.  Закомментированная логика из C#-версии была правильно проигнорирована при портировании.
