# План портирования json2.js

Файл `json2.js` - это сторонняя библиотека (полифил) для работы с JSON.

## Функциональность

*   **Назначение**: Предоставить реализацию глобального объекта `JSON` с методами `stringify` и `parse` для старых браузеров (в частности, старых версий Internet Explorer), в которых не было встроенной поддержки JSON.
*   **Реализация**: Это стандартная, общеизвестная библиотека `json2.js` от Дугласа Крокфорда.

## Проверка на существующую реализацию в Android

- **Результат:** Неприменимо.
- **Объяснение**: Современный компонент `WebView` в Android основан на Chromium и имеет полную, быструю, нативную реализацию объекта `JSON`. Этот полифил абсолютно не нужен.

## Решение для портирования на Android

Никаких действий не требуется. Файл не нужно никуда копировать или использовать.
