# План портирования ForumTopicJs.cs

Файл `ForumTopicJs.cs` - это часть `partial` класса `Filter`, отвечающая за модификацию скрипта, отображающего темы форума.

## Функциональность в C#

*   **Назначение**: Реализовать опцию "легкого форума", удаляя аватарки пользователей для экономии трафика и ускорения загрузки.
*   **Логика**: Если в профиле пользователя включена опция `LightForum`, фильтр находит и удаляет из JavaScript-кода две строки, отвечающие за добавление тегов `<img>` для аватарок.
