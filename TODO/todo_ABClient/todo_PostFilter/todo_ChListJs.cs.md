
### 1. План портирования ChListJs.cs

Файл `ChListJs.cs` является фильтром для скрипта `ch_list.js`, который отвечает за отображение списка пользователей в чате.

### 2. Функциональность в C#

- **Назначение:** Заменить стандартный `ch_list.js` от сервера на кастомную, модифицированную версию из ресурсов клиента.
- **Логика:**
    1.  Код полностью игнорирует содержимое файла, полученного от сервера.
    2.  Он берет локальный файл `Resources.ch_list` (который является файлом `ABClient/ch_list.js`).
    3.  Выполняет в нем простую замену: `Replace("alt=", "title=")`. Это, вероятно, необходимо для корректного отображения всплывающих подсказок в элементе `WebBrowser`.
    4.  Возвращает измененный локальный файл.

### 3. Решение для портирования на Android

Необходимо воспроизвести поведение полной замены файла. Файл `ch_list.js` уже присутствует в `assets` Android-проекта.

- **Архитектура:** Логика будет реализована в методе `process()` класса `postfilter.ChListJs.java`.
