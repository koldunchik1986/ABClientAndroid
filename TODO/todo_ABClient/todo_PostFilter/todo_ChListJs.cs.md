# План портирования ChListJs.cs

Файл `ChListJs.cs` - это часть `partial` класса `Filter`, отвечающая за модификацию скрипта `ch_list.js`.

## Функциональность в C#

*   **Назначение**: Модифицировать на лету JavaScript-файл `ch_list.js`.
*   **Логика**: Загружает содержимое `ch_list.js` из ресурсов и заменяет все вхождения `alt=` на `title=`, чтобы корректно отображались всплывающие подсказки.
