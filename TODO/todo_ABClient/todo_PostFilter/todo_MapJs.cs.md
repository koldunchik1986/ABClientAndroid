# План портирования MapJs.cs

Файл `MapJs.cs` - это часть `partial` класса `Filter`, отвечающая за подмену скрипта карты `map.js`.

## Функциональность в C#

*   **Назначение**: Полностью заменить серверную версию `map.js` на локальную, которая хранится в ресурсах приложения.
*   **Логика**: Метод `MapJs` игнорирует входящий массив байт и просто возвращает содержимое файла `Resources.map`.
*   **Закомментированная логика**: В файле также есть закомментированный код, который предполагал не полную замену файла, а его модификацию для внедрения вызовов `window.external` с целью получения размеров карты из настроек клиента.

## Проверка на существующую реализацию в Android
