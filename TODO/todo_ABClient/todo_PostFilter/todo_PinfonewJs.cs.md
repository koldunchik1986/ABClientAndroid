# План портирования PinfonewJs.cs

Файл `PinfonewJs.cs` - это часть `partial` класса `Filter`, отвечающая за модификацию скрипта `pinfo_v01.js`.

## Функциональность в C#

*   **Назначение**: Изначально предназначался для внедрения в страницу информации о персонаже кастомных всплывающих подсказок для предметов и для подмены изображений персонажей.
*   **Закомментированная логика**: Весь код в методе закомментирован. Он должен был добавлять `onmousemove` и `onmouseout` к тегам `<img>` предметов, чтобы вызывать JS-функции `showslo()` и `hideslo()`. Также содержал логику для подмены аватарок через `window.external.ObrazSubstitute`.
*   **Текущее состояние**: Фильтр отключен и ничего не делает.

## Проверка на существующую реализацию в Android

- **Результат:** Функциональность корректно реализована как "пустышка".
- **Объяснение**:
    1.  В проекте существует класс `app/src/main/java/ru/neverlands/abclient/postfilter/PinfoJs.java`.
    2.  Его метод `process()` просто возвращает исходный массив байт.
    3.  Комментарии в файле подтверждают, что оригинальная логика была закомментирована и поэтому не была портирована.

## Решение для портирования на Android

Никаких действий не требуется. Это часть заброшенной в C#-версии функциональности по созданию кастомных тултипов.
