### 1. План портирования MapActAjaxPhp.cs

Файл `MapActAjaxPhp.cs` является фильтром для `map_act_ajax.php`, который, предположительно, обрабатывает результаты действий на карте (копание, сбор ресурсов и т.д.).

### 2. Функциональность в C#

- **Назначение:** Обработка AJAX-ответов от `map_act_ajax.php`.
- **Логика:** **Полностью отсутствует.** Метод-фильтр является "заглушкой" (stub). Он преобразует массив байт в строку и тут же возвращает исходный, немодифицированный массив байт.

```csharp
private static byte[] MapActAjaxPhp(byte[] array)
{
    var html = AppVars.Codepage.GetString(array);
    return array;
}
```

### 3. Решение для портирования на Android

Поскольку в оригинальном коде нет никакой логики, портирование сводится к созданию аналогичной заглушки в Java.

- **Архитектура:** Логика (точнее, ее отсутствие) будет находиться в `postfilter.MapActAjaxPhp.java`.
