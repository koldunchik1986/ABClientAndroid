
### 1. План портирования MainPhpTied.cs

Файл `MainPhpTied.cs` содержит простую вспомогательную функцию для парсинга количества "привязанных" вещей из HTML-кода страницы.

### 2. Функциональность в C#

- **Назначение:** Извлечь количество привязанных вещей и обновить соответствующий элемент интерфейса.
- **Логика:**
    1.  Функция получает на вход HTML-код и позицию, с которой начинается числовое значение.
    2.  Извлекает и парсит это число.
    3.  Вызывает делегат `UpdateTied` на главной форме (`AppVars.MainForm`), чтобы обновить UI, отображающий это количество.

### 3. Решение для портирования на Android

Это вспомогательная функция, которая не модифицирует HTML, а только читает из него данные для обновления UI. Ее логика должна быть воспроизведена в соответствующем парсере на стороне Android.

### 4. План реализации

1.  **Дополнить класс `PlayerStateParser` в Android:**
    - Этот класс будет отвечать за извлечение всей информации о состоянии персонажа со страниц.
2.  **Реализовать метод `parseTiedItemsCount(html)`:**
    - Метод будет искать в HTML-коде и извлекать количество привязанных вещей.
    - Возвращать это значение.
3.  **Обновление UI:**
    - Компонент, отвечающий за отображение состояния персонажа, будет вызывать `parseTiedItemsCount` и обновлять соответствующий `TextView`.
