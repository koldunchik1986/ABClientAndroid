# План портирования BuildingJs.cs

Файл `BuildingJs.cs` - это часть `partial` класса `Filter`, отвечающая за модификацию скрипта `building.js`.

## Функциональность в C#

*   **Назначение**: Модифицировать на лету JavaScript-файл `building.js`.
*   **Логика**: Принимает содержимое файла. Если в профиле пользователя включена опция `ChatKeepMoving`, то из скрипта удаляется вызов функции `parent.clr_chat();`, чтобы предотвратить очистку чата при входе в здания.
