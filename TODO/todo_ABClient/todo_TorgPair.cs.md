# План портирования TorgPair.cs

Файл `TorgPair.cs` определяет простой класс-модель данных для одного правила в системе торга.

## Функциональность в C#

*   **Назначение**: Хранить одну запись из таблицы торга, которая состоит из диапазона цен и бонуса.
*   **Свойства (Properties)**:
    *   `PriceLow`: `int` - Нижняя граница диапазона цен.
    *   `PriceHi`: `int` - Верхняя граница диапазона цен.
    *   `Bonus`: `int` - Бонус, применяемый к ценам в этом диапазоне.
