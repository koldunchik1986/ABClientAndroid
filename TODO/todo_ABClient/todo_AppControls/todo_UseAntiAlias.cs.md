# План портирования UseAntiAlias.cs

Файл `UseAntiAlias.cs` - это небольшой класс-хелпер для временного включения сглаживания.

## Функциональность в C#

*   **Назначение**: Упростить управление состоянием сглаживания (`AntiAlias`) для объекта `Graphics`. Использует паттерн `IDisposable` для автоматического восстановления предыдущего состояния после завершения блока `using`.
*   **Реализация**: 
    *   В конструкторе сохраняет текущий `SmoothingMode` и устанавливает новый (`SmoothingMode.AntiAlias`).
    *   В методе `Dispose()` восстанавливает сохраненный `SmoothingMode`.

## Решение для портирования на Android

Прямое портирование класса не имеет смысла, так как он решает проблему, специфичную для `GDI+`. В Android сглаживание - это флаг на объекте `Paint`, а не на `Canvas`.

Когда потребуется рисовать со сглаживанием в кастомном `View` (в методе `onDraw`), нужно будет сделать следующее:

```java
Paint paint = new Paint();
paint.setAntiAlias(true);
// или
paint.setFlags(paint.getFlags() | Paint.ANTI_ALIAS_FLAG);

canvas.drawCircle(..., paint);
```

Поскольку это всего одна строка, создание отдельного класса-хелпера, подобного `UseAntiAlias`, избыточно. Разработчик просто устанавливает флаг на том `Paint` объекте, который он использует для рисования.
