# План портирования UseClearTypeGridFit.cs

Файл `UseClearTypeGridFit.cs` - это небольшой класс-хелпер для временного изменения настроек рендеринга текста.

## Функциональность в C#

*   **Назначение**: Упростить управление состоянием рендеринга текста (`TextRenderingHint`) для объекта `Graphics`. Использует паттерн `IDisposable` для автоматического восстановления предыдущего состояния.
*   **Реализация**: 
    *   В конструкторе сохраняет текущий `TextRenderingHint` и устанавливает новый (`TextRenderingHint.SystemDefault`).
    *   В методе `Dispose()` восстанавливает сохраненный `TextRenderingHint`.

## Проверка на существующую реализацию в Android

- **Результат:** Неприменимо.
- **Объяснение**: Этот класс решает задачу, специфичную для `GDI+`. В Android качество рендеринга текста управляется флагами на объекте `Paint`.

## Решение для портирования на Android

Никаких действий не требуется. Когда потребуется управлять качеством рендеринга текста в кастомном `View`, нужно будет установить соответствующие флаги на объекте `Paint`, который используется для рисования этого текста.

```java
Paint textPaint = new Paint();
textPaint.setFlags(textPaint.getFlags() | Paint.SUBPIXEL_TEXT_FLAG);

canvas.drawText(..., textPaint);
```

Создание отдельного класса-хелпера для этого нецелесообразно.
