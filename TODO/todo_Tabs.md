# Анализ папки Tabs

В этой папке содержатся классы, связанные с работой вкладок (табов) в главном окне приложения.

## Список файлов и статус анализа

| Файл                  | Описание                                      | Статус подробного анализа (по rules_MDcreate.md) |
| --------------------- | --------------------------------------------- | --------------------------------------------------- |
| `TabClass.cs`           | Класс, описывающий одну вкладку.              | `[x]` Проанализирован                                 |
| `TabType.cs`            | Перечисление (enum) типов вкладок.            | `[x]` Проанализирован                                 |
| `TcHitTestFlags.cs`     | Перечисление (enum) для определения клика.    | `[x]` Проанализирован                                 |
| `TcHitTestInfo.cs`      | Структура для определения клика по вкладке.   | `[x]` Проанализирован                                 |
