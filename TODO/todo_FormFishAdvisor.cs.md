# План портирования FormFishAdvisor.cs

Файл `FormFishAdvisor.cs` реализует окно "Советник по рыбалке" — инструмент, который помогает игроку выбрать наиболее прибыльное место для рыбалки в зависимости от его навыков.

## Функциональность в C#

*   **База данных**: В коде жестко закодирован большой список (`List<FishTip>`), представляющий собой базу данных советов. Каждый совет содержит информацию о прибыльности, требуемом умении, координатах локации, обитающих там ботах и типе рыбы/наживки.
*   **UI**: Форма предоставляет пользователю два поля для ввода: текущее умение рыбалки и максимальный уровень ботов, с которыми он готов сражаться.
*   **Логика**: При нажатии кнопки "Рассчитать", программа фильтрует свою базу данных, оставляя только те советы, которые подходят под указанные умение и уровень. Затем отфильтрованный список сортируется (вероятно, по прибыльности) и выводится в виде таблицы.

## План портирования на Android

Это полезный инструмент, который следует воссоздать с использованием нативных компонентов Android.

1.  **Управление данными**:
    *   **Создать `FishTip.java`**: Создать POJO-класс, который будет содержать все поля из C#-аналога (`money`, `fishUm`, `location`, `maxBotLevel`, `botDescription`, `tip`).
    *   **Создать `assets/fish_tips.json`**: Вынести жестко закодированную базу данных из кода в отдельный JSON-файл. Это сделает код чище и позволит в будущем обновлять базу советов без перекомпиляции приложения.
    *   **Создать `FishTipRepository.java`**: Создать класс-синглтон, который при первом обращении будет загружать и парсить `fish_tips.json` (с помощью библиотеки `Gson`) в `List<FishTip>` и кэшировать его в памяти.

2.  **UI (`FishAdvisorActivity.java`)**:
    *   Создать новую `Activity` для советника.
    *   В XML-разметке разместить:
        *   `com.google.android.material.textfield.TextInputEditText` для ввода умения рыбалки.
        *   `android.widget.Spinner` для выбора максимального уровня ботов.
        *   `com.google.android.material.button.MaterialButton` с текстом "Рассчитать".
        *   `androidx.recyclerview.widget.RecyclerView` для отображения списка результатов.

3.  **Логика (`FishAdvisorViewModel.java`)**:
    *   Создать `ViewModel` для этого экрана.
    *   При нажатии на кнопку "Рассчитать":
        1.  Получить значения из `EditText` и `Spinner`.
        2.  Запросить у `FishTipRepository` полный список советов.
        3.  С помощью `Stream API` (требуется `minSdkVersion 24`) или обычного цикла отфильтровать список по заданным критериям.
        4.  Отсортировать отфильтрованный список с помощью `Collections.sort()` (для этого `FishTip.java` должен реализовывать `Comparable`).
        5.  Обновить `LiveData<List<FishTip>>`, на который подписан `RecyclerView.Adapter`, чтобы отобразить результаты.
