# Анализ папки Lez

В этой папке содержатся классы, связанные с системой "Автобой" (АБ), включая логику боя, настройки и управление ботами и заклинаниями.

## Список файлов и статус анализа

| Файл                         | Описание                                               | Статус подробного анализа (по rules_MDcreate.md) |
| ---------------------------- | ------------------------------------------------------ | --------------------------------------------------- |
| `FormSettingsAb.cs`          | Форма для настроек Автобоя.                            | `[x]` Проанализирован                                 |
| `FormSettingsAb.Designer.cs` | Код дизайнера для формы `FormSettingsAb`.              | `[x]` Проанализирован                                 |
| `FormSettingsAb.resx`        | Ресурсы для формы `FormSettingsAb`.                    | `[x]` Проанализирован                                 |
| `LezBotsClass.cs`            | Класс, описывающий одного бота.                        | `[+]` Реализован                                 |
| `LezBotsClassCollection.cs`  | Коллекция для `LezBotsClass`.                          | `[+]` Реализован                                 |
| `LezBotsGroup.cs`            | Класс, описывающий группу ботов.                       | `[+]` Реализован                                 |
| `LezFight.cs`                | Основной класс с логикой ведения боя.                  | `[+]` Реализован                                 |
| `LezNode.cs`                 | Узел (вероятно, для дерева принятия решений).          | `[+]` Реализован                                 |
| `LezSpell.cs`                | Класс, описывающий одно заклинание.                    | `[+]` Реализован                                 |
| `LezSpellCollection.cs`      | Коллекция для `LezSpell`.                              | `[+]` Реализован                                 |
