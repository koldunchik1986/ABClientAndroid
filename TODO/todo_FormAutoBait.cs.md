# План портирования FormAutoBait.cs

Файл `FormAutoBait.cs` и его `.designer.cs` описывают простое диалоговое окно для запуска функции "Автоприманка ботов".

## Функциональность в C#

*   **UI**: Форма содержит текстовое поле для ввода количества использований приманки и кнопку "Запустить приманки!".
*   **Назначение**: Позволяет пользователю указать, сколько раз подряд клиент должен автоматически использовать приманку (вероятно, на ботов в локации, чтобы спровоцировать их на бой).
*   **Логика**: Сама форма не содержит логики. Она просто предоставляет UI, а основная логика цикла, вероятно, запускается и управляется из `FormMain` после нажатия на кнопку.

## План портирования на Android

Создание отдельного модального окна для этой простой настройки избыточно. Эту функциональность следует интегрировать в общий экран настроек приложения, в раздел, посвященный рыбалке или бою.

1.  **UI Настроек (`PreferenceScreen`)**:
    *   В файле `res/xml/root_preferences.xml` (или в отдельном экране настроек рыбалки) создать категорию "Автоприманка".
    *   Добавить `SwitchPreferenceCompat` для включения/выключения функции:
        ```xml
        <SwitchPreferenceCompat
            app:key="autobait_enabled"
            app:title="Автоприманка ботов"
            app:summary="Автоматически использовать приманку указанное количество раз" />
        ```
    *   Добавить `EditTextPreference` для указания количества:
        ```xml
        <EditTextPreference
            app:key="autobait_count"
            app:title="Количество использований"
            app:defaultValue="9999"
            android:inputType="number"
            app:dependency="autobait_enabled" />
        ```
        Атрибут `app:dependency="autobait_enabled"` сделает этот пункт неактивным, пока `SwitchPreferenceCompat` выключен.

2.  **Логика**:
    *   Основная логика клиента (вероятно, часть будущего `MainPhpProcessor`), отвечающая за автоматические действия, должна будет считывать эти два значения из `SharedPreferences`.
    *   Если `autobait_enabled` имеет значение `true`, то при определенных условиях (например, при входе в локацию с ботами) должен запускаться цикл, который будет инициировать использование приманки `autobait_count` раз.
