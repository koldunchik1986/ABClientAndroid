| `Dice.cs`        | Генератор случайных чисел.             | `[x]` Проанализирован                                 |
| `Http.cs`        | Вспомогательные функции для HTTP.      | `[x]` Проанализирован                                 |
| `Misc.cs`        | Различные вспомогательные функции.     | `[x]` Проанализирован                                 |
| `NativeMethods.cs`| Импорт нативных функций WinAPI.        | `[x]` Проанализирован                                 |
| `Pack.cs`        | Функции для работы с архивами.         | `[x]` Проанализирован                                 |
| `Russian.cs`     | Функции для работы с русской кодировкой.| `[x]` Проанализирован                                 |
