# План портирования Http.cs

Файл `Http.cs` в папке `Helpers` является набором статических утилит для низкоуровневой работы с HTTP.

## Функциональность в C#

Этот класс практически полностью дублирует функциональность класса `Utilities.cs` из папки `ABProxy`. Он содержит те же самые методы для:

*   **Декомпрессии**: `GzipExpand`, `DeflateExpand`.
*   **Обработки `chunked`-ответов**: `DoUnchunk`, `IsChunkedBodyComplete`.
*   **Парсинга строк с хостом и портом**.

## Решение для портирования на Android

**Портировать этот класс не нужно.**

Как и в случае с `Utilities.cs`, вся его функциональность является стандартной для современных HTTP-библиотек и системных классов Android.

*   Декомпрессия и обработка `chunked` encoding выполняются библиотекой `OkHttp` автоматически.
*   Парсинг URL и его компонентов выполняется классом `android.net.Uri`.

Этот класс, скорее всего, является устаревшим дубликатом `Utilities.cs` и не несет никакой уникальной логики, требующей переноса.
