| `AssemblyInfo.cs`     | Информация о сборке.                          | `[ ]` Не проанализирован                              |
| `Resources.Designer.cs`| Дизайнер ресурсов.                            | `[ ]` Не проанализирован                              |
| `Resources.resx`      | Ресурсы проекта.                              | `[ ]` Не проанализирован                              |
| `Settings.Designer.cs`| Дизайнер настроек.                            | `[ ]` Не проанализирован                              |
| `Settings.settings`   | Настройки проекта.                            | `[ ]` Не проанализирован                              |
