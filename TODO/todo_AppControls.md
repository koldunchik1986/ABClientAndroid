# Общий анализ папки AppControls

Папка `AppControls` содержит набор кастомных UI-компонентов (контролов), которые расширяют или заменяют стандартные элементы Windows Forms для создания более богатого и функционального пользовательского интерфейса.

## Содержимое папки

*   **`ExtendedWebBrowser.cs`**: Вероятно, самый важный класс в этой папке. Это наследник стандартного `WebBrowser`, который позволяет добавлять кастомную логику, перехватывать события навигации, управлять DOM-документом и т.д. Это "сердце" отображения игрового мира.
*   **`ListBoxEx.cs`, `TreeViewEx.cs`**: Расширенные версии стандартных компонентов для отображения списков и деревьев. Могут содержать улучшенную отрисовку, поиск и т.д.
*   **`CollapsibleSplitter.cs`**: Кастомный разделитель панелей, который можно сворачивать/разворачивать.
*   **`Office2007ColorTable.cs`, `Office2007Render.cs`**: Классы, отвечающие за отрисовку элементов интерфейса в стиле Microsoft Office 2007. Это определяет внешний вид приложения.
*   **`Use...` классы**: Утилиты для управления качеством рендеринга графики (например, сглаживание текста).

## Решение для портирования на Android

**Прямое портирование этих компонентов невозможно.** Они полностью завязаны на архитектуру и API Windows Forms.

Вместо этого, необходимо воссоздать их функциональность с использованием нативных инструментов Android:

*   **`ExtendedWebBrowser`** -> Его функциональность будет распределена между `android.webkit.WebView`, кастомным `WebViewClient` и `WebChromeClient`.
*   **`ListBoxEx`, `TreeViewEx`** -> Заменяются на `RecyclerView` с кастомными `Adapter` и `ViewHolder`. `RecyclerView` — это мощный и гибкий компонент для отображения любых списков.
*   **`CollapsibleSplitter`** -> Подобное поведение можно реализовать с помощью `MotionLayout` или кастомной анимации, управляющей видимостью и размерами `View`.
*   **Стилизация Office 2007** -> Полностью игнорируется. Вместо нее для Android-приложения будет использоваться **Material Design** — стандартный язык дизайна для Android, который обеспечит современный и привычный для пользователей внешний вид.
