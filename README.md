# ABClient для Android

Это Android-версия клиента автоматизации для браузерной игры neverlands.ru. Приложение сохраняет 100% логику и функциональность оригинального ПК-приложения ABClient.

## Особенности

- Полная поддержка всех функций оригинального ABClient
- Локальный прокси-сервер для перехвата и модификации HTTP-трафика
- Автоматизация игровых действий (рыбалка, сбор трав, добыча руды и т.д.)
- Сохранение сессии по полученным куки
- Поддержка внешних прокси-серверов
- Логирование HTTP-запросов и ответов
- Кэширование игровых ресурсов

## Требования

- Android 5.0 (API 21) или выше
- Доступ в интернет

## Установка

1. Скачайте последнюю версию APK из раздела [Releases](https://github.com/username/ABClientAndroid/releases)
2. Установите APK на ваше устройство
3. Разрешите приложению доступ в интернет

## Использование

1. Запустите приложение
2. Введите ваш логин и пароль от игры neverlands.ru
3. Нажмите кнопку "Войти"
4. После успешного входа вы увидите игровой интерфейс
5. Используйте боковое меню для навигации по разделам игры

## Настройки

В приложении доступны следующие настройки:

### Основные настройки
- Запрашивать подтверждение при выходе
- HTTP-логирование
- Текстовое логирование
- Показывать производительность
- Обновлять кэш
- Очистить кэш
- Очистить логи

### Настройки прокси
- Использовать прокси
- Адрес прокси
- Имя пользователя прокси
- Пароль прокси

### Автоматизация
- Автоматическая рыбалка
- Автоматический сбор трав
- Автоматическая добыча руды
- Автоматическая рубка деревьев
- Автоматическое копание
- Автоматическая торговля
- Активность торговли

## Разработка

Проект разработан с использованием:
- Java для основной логики
- Android SDK
- OkHttp для HTTP-запросов
- WebView для отображения игрового интерфейса

## Структура проекта

- `ru.neverlands.abclient` - основной пакет приложения
  - `ABClientApplication.java` - класс приложения
  - `MainActivity.java` - основная активность
  - `LoginActivity.java` - активность входа
  - `SettingsActivity.java` - активность настроек
- `ru.neverlands.abclient.model` - модели данных
  - `UserConfig.java` - конфигурация пользователя
- `ru.neverlands.abclient.proxy` - прокси-сервер
  - `ProxyService.java` - сервис прокси
  - `SessionHandler.java` - обработчик сессий
  - `Cache.java` - кэш
  - `CookiesManager.java` - менеджер куки
- `ru.neverlands.abclient.postfilter` - фильтры HTTP-ответов
  - `Filter.java` - основной класс фильтрации
- `ru.neverlands.abclient.utils` - утилиты
  - `AppVars.java` - глобальные переменные
  - `DataManager.java` - менеджер данных
  - `Log.java` - логирование
  - `Russian.java` - работа с русской кодировкой
  - `VersionClass.java` - информация о версии

## Лицензия

Этот проект является портом оригинального ABClient и распространяется под той же лицензией.

## Предупреждение

Использование данного приложения может нарушать правила игры neverlands.ru. Используйте на свой страх и риск.
